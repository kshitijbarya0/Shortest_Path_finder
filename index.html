How to Use:

Welcome to the Shortest Path Finder web app! Follow these steps to get started:

  1. Click on the canvas to add nodes.
  2. Enter the source node index, destination node index, and the distance between them.
  3. Click the "Add Edge" button to add the edge between the nodes.
  4. Repeat steps 2 and 3 to add more edges.
  5. Specify the source and destination nodes in the controls section.
  6. Click the "Find Shortest Path" button to calculate the shortest path.
